# Permutation.cpp
This C++ program calculates the Permutation (ⁿPᵣ) of two numbers using the factorial method. It takes the values of n and r as input, computes the factorial of n and (n - r) using a user-defined function, and then applies the permutation formula ⁿPᵣ = n! / (n − r)! to display the result. This program use of functions, loops, arithmetic operations.
